To run this project , make sure you have xampp server installed 

make a file in C:\Xampp\htdocs and paste all these files 

configure the file name in config.php 

Create a database and import the the url_shortner.sql in the database 

configure the database name in the config.php file 

Done! open your xampp and mysql server and run the project. 

Make sure you have internet connection as well 

Enjoy!!!
